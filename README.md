## Comandos
|Comando         |Descrição                      |Permissão                    |
|----------------|-------------------------------|-----------------------------|
|/cmd        |Esse será o comando definido na config para o canal|`yuchat.channel.<nomedocanal>`           |
|/yuchat reload    |Recarrega o plugin por completo|`yuchat.admin`       |
|/chatclear |Limpa o chat de todos os jogadores	     |`yuchat.admin`			   |
|/tell |Envia um mensagem privada para algum jogador|`N/D`			   |
|/r |Responde o jogador que enviou uma mensagem privada|`N/D`			   |
|/espiar |Veja as mensagens de todos os jogadores do servidor    |`yuchat.espiar`			   |

## Sobre
Plugin feito especialmente para OlympusMC - 1.16.4
